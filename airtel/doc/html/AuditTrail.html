Audit Trail

Main Tasks ››



Vital activities carried out on the portal are logged. The log reports showing the activities of each system user are available to users with the right permissions to do that.


Audit Trail module provides access to system logs. The audit reports are date based.



HOW TO DISPLAY AUDIT TRAIL REPORTS

  1. Click on the Audit Trail menu item under Main Tasks menu.
  2. Select Report Start Date and Report End Date. To display reports of a single day's activities, the Report Start Date and Report End Date selection are made to be the same.
  3. Click on Display Report button. Sample of display Audit Trail report is shown in the picture below.



NOTE: The fetched audit records are displayed in View Report tab. Report Start Date and Report End Date are displayed in the Report Parameters tab.
