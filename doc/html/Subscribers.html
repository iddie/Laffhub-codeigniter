Subscribers

Main Tasks ››


The subscribers module displays the names of subscribers and their subscription status.


HOW TO VIEW SUBSCRIBERS

  1. Click on the Subscribers menu item under Main Tasks menu. The default network is loaded (if set in the Portal Settings under Settings/Users menu). Records of subscribers in the default network are displayed in the table.
  2. To display subscribers from networks other than the default network, select Network button and click on Display button.



NOTE: If no network is selected, all the available subscribers from all the networks will be displayed when the Display button is clicked.
