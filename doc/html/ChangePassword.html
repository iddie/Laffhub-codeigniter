Change Password

Edit User Information ››


Portal users can change their access password through the Change Password module. To change password a user must first sign in.


Users can edit the following fields: Old Password, New Password and Confirm Password.


All the editable fields are mandatory. That is, valid values must be entered before you can successfully change the access password.


HOW TO CHANGE USER PASSWORD

  1. Click on the Change Password menu item under Edit User Information menu.
  2. Enter the current password (the Old Password you want to change).
  3. Enter the New Password.
  4. Enter the New Password a second time to confirm your entry.
  5. Click on Change Password button. A confirmation prompt (below) appears.


  1. Click on OK button to continue with the change or CANCEL button to abort the action.
